# 项目介绍

Lithe Admin 是一个轻量级、优雅的 Vue 3 后台管理模板，专注于提供灵活的主题定制、低耦合的架构设计和渐进式增强的开发体验。

## 特性

- **现代技术栈** - 基于 Vue 3.5 + Vite 7 + TypeScript 5.9 构建
- **优雅的 UI** - 使用 Naive UI 2.43 组件库，提供精美的界面
- **灵活主题** - 支持亮色/暗色模式，可自定义主题色和视觉效果
- **动态路由** - 基于权限的动态路由系统，支持菜单级权限控制
- **响应式布局** - 完善的移动端适配，支持多种导航模式
- **丰富组件** - 内置数据表格、图表、拖拽等常用功能组件
- **开发友好** - TypeScript 类型支持，ESLint + Prettier 代码规范

## 技术栈

| 技术        | 版本  | 说明                                |
| ----------- | ----- | ----------------------------------- |
| Vue         | 3.5+  | 渐进式 JavaScript 框架              |
| Vite        | 7.x   | 下一代前端构建工具（使用 Rolldown） |
| TypeScript  | 5.9+  | JavaScript 的超集                   |
| Naive UI    | 2.43+ | Vue 3 组件库                        |
| TailwindCSS | 4.x   | 原子化 CSS 框架                     |
| Pinia       | 2.x   | Vue 状态管理库                      |
| Vue Router  | 4.x   | Vue 官方路由                        |
| ECharts     | 5.x   | 数据可视化图表库                    |
